* 포인터 변수 없이 number 변수의 주소 값을 아래와 같이 출력하세요. 

`int number = 5983;`

```

```

* 아래 변수를 포인터 변수를 통해서 아래와 같이 출력 하세요.  

`int number = 15;`

> 출력 예시 

```
주소 : 0x7ffeefbff49c
```
